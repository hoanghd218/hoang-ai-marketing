description: Facebook and YouTube use Vietnamese, while LinkedIn, X.com, and Instagram use English
type: feedback

Only Facebook and YouTube content should be written in Vietnamese. LinkedIn, X.com (Twitter), and Instagram content must be written in English.

**Why:** User's distribution strategy — Vietnamese for local reach (Facebook, YouTube), English for global professional audience (LinkedIn, X.com, Instagram).

**How to apply:** When generating content for LinkedIn, X.com, or Instagram — always write in English. When generating for Facebook or YouTube — write in Vietnamese with full diacritics. Apply this rule in all content creation skills including mkt-repurpose-to-social, mkt-linkedin-post-creator, mkt-carousel-creator, mkt-xcom-post-creator.
