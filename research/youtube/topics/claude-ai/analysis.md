## 3. Top Insights (7-8 cái gai nhất)

### Insight 1 — Claude Code source code rò rỉ hóa ra cực kỳ đơn giản
Chỉ là vòng lặp prompt + regex phát hiện user tức giận + đôi khi fake tools khi user đang reverse-engineer. Phá vỡ huyền thoại "AI phức tạp".
**Nguồn:** Claude Code Leak (Alberta Tech)

### Insight 2 — Thư mục markdown phẳng đánh bại RAG/Vector DB
Wiki markdown files giảm 95% token so với vector DB, setup chỉ mất 5 phút, chạy local — ưu việt ở quy mô nhỏ-vừa.
**Nguồn:** Karpathy 10x Claude Code

### Insight 3 — Copy-trade chính trị gia Mỹ đánh bại S&P 500 2.2x
Claude + Alpaca + Capital Trades tự động track + clone trades của congressmen (họ có insider info hợp pháp).
**Nguồn:** Claude Changed Stock Market

### Insight 4 — Motion graphics bằng code thay vì After Effects
Claude Code + Remotion (ReactJS) render video graphics — biến công việc UI-based thành coding-based.
**Nguồn:** Claude Code Changed YouTube Videos Forever

### Insight 5 — Khai tử Notion/Obsidian bằng 1 folder
Không cần app, chỉ cần folder + Claude tự build SQLite DB + web UI custom — "trả quyền về user".
**Nguồn:** Claude killed Note-Taking Apps

### Insight 6 — AI đọc DOM trade chart, không screenshot
TradingView MCP để Claude đọc mã nguồn HTML realtime (1-second candles) thay vì nhìn ảnh — độ chính xác 100%.
**Nguồn:** Claude to Trading View

### Insight 7 — Biến Claude thành gia sư English qua Personal Preferences
Set preference: "chèn từ vựng nâng cao + giải nghĩa". Học thụ động mỗi lần chat — micro-learning hack.
**Nguồn:** Full Claude Tutorial (Ayushman)

### Insight 8 — Đánh đổi quyền riêng tư hộp thư
Connectors cho Claude đọc toàn bộ email. Tiện nhưng AI nắm "từng từ" của mọi thư — nỗi sợ khi phụ thuộc hoàn toàn.
**Nguồn:** Claude Knows My Inbox

## 4. Gaps & Cơ hội Content (cho Hoang)

### Gap 1 — Ứng dụng cho SME Việt Nam & bán lẻ truyền thống
Tất cả video target solopreneur/creator/trader Mỹ. **Cơ hội:** Claude cho cửa hàng F&B, e-commerce local, quản lý kho, CSKH chuẩn tiếng Việt.

### Gap 2 — Team Collaboration (không phải solo)
Toàn bộ tôn sùng "personal AI assistant". **Cơ hội:** Hướng dẫn team 5-10 người dùng Claude Projects chung, prompt library cho team Marketing, Claude Team/Enterprise workflow.

### Gap 3 — Bản địa hóa tiếng Việt
Hầu hết bằng English (1 video Hindi). **Cơ hội:** Claude xử lý tiếng Việt vượt trội ChatGPT, phân tích user behavior VN, Artifacts hoàn toàn tiếng Việt.

### Gap 4 — Misconception "AI là phép màu"
Video demo kết quả hoàn hảo, không show khi Claude fail/bug/hallucinate. **Cơ hội:** "Mặt trái của Claude Code" — debugging, giới hạn permissions, khi nào Claude sai.

### Gap 5 — Data privacy & guardrails cho doanh nghiệp
Video khuyến khích cấp full access dễ dãi. **Cơ hội:** Setup rào chắn bảo mật, local LLM fallback, không leak data lên Anthropic servers.

### Gap 6 — Hidden costs khi scale
Khen 95% token efficiency nhưng không nói về rate limits ($20 Pro/$200 Max), token optimization. **Cơ hội:** Bài toán kinh tế Claude cho SME — Pro vs API vs Max.

### Gap 7 — Kill-switch cho tự động hóa tài chính
Video trading bot không đề cập slippage, API fail, legal risk khi AI control tiền thật. **Cơ hội:** Human-in-the-loop cho automation tiền bạc/pháp lý.
