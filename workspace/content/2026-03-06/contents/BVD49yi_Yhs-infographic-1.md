# GPT 5.4 vs Opus 4.6 — Chọn model nào cho việc gì?

**Slide 1 (Hook):**
GPT 5.4 hay Opus 4.6?
Câu trả lời phụ thuộc vào bạn đang làm gì.

**Slide 2: Framework 4 chiều đánh giá**
Đừng đánh giá AI chỉ bằng benchmark.
Test 4 chiều thực tế:
- Viết sáng tạo
- Giải thích / dạy học
- Phân tích / lập luận
- Visual reasoning

**Slide 3: Kết quả test thực tế**
Viết sáng tạo — Opus thắng (ngắn gọn, có chiều sâu)
Giải thích khái niệm — Opus thắng (rõ ràng, dễ hiểu)
Phân tích địa chính trị — Opus thắng (quyết đoán, theo đúng chỉ dẫn)
Visual reasoning — GPT 5.4 thắng (chính xác hơn với hình ảnh)

**Slide 4: Chọn model theo công việc**
Dùng GPT 5.4 khi: Coding, debugging, game dev, xử lý hình ảnh
Dùng Opus 4.6 khi: Viết content, giải thích, tạo Workflow, AI agent tổng hợp

**Slide 5: Nguyên tắc chọn model**
Không có model "tốt nhất". Chỉ có model "phù hợp nhất".
Test với công việc thật của bạn. Đó là cách duy nhất để biết.
