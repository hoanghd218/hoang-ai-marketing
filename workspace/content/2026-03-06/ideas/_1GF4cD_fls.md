## Buoc 2: Knowledge Nuggets

1. **Tool fragmentation la rao can lon nhat** -- Xay AI Agent hien tai doi hoi nhay giua 5+ tool khac nhau, viet glue code, va "cau troi khong hong."

2. **Mo ta bang ngon ngu tu nhien thay vi code** -- Mo hinh moi: thay vi viet code de build agent, ban chi can mo ta bang tieng Anh nhung gi ban muon.

3. **No-code to pro-code spectrum** -- Nen tang phuc vu ca nguoi moi (no-code), nguoi trung binh (low-code), va developer (pro-code voi Python, API).

4. **Security va governance phai co tu dau** -- Nhieu team xay AI agent truoc roi moi lo DLP policies, access controls, audit trails sau. Day la sai lam lon tao "governance debt."

5. **3 lop cua AI Agent platform** -- Build (visual builder), Govern (bao mat va kiem soat), Deploy (thu vien agent san co).

6. **Thu vien 3,000+ pre-built agents** -- Khong can xay tu dau, co the bat dau tu cac agent da duoc xay san.

### Draft 3: Facebook Post -- Framework "3 lop xay AI Agent khong phai developer"

**[HOOK]**

Moi lop quan trong nhu nhau. Thieu 1 lop la he thong se sup.

**[LAST DAB + CTA]**
Build agent khong kho. Build agent an toan va ben vung moi kho. Bat dau tu 3 lop nay.
