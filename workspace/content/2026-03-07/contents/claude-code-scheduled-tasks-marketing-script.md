## 🔍 Chi Tiết Áp Dụng

### 1. 💃 The Dance
- **Số conflict loops:** 6
- **Trích dẫn:**
  > Loop 1: [Context: bạn mất 5-10 giờ/tuần cày YouTube] → NHƯNG giờ có cách khác — Scheduled Tasks
  > Loop 2: [Context: setup dễ, 4 bước] → NHƯNG setup dễ không nghĩa bạn hiểu sức mạnh thực sự
  > Loop 3: [Context: hầu hết nghĩ automation = code cố định] → NHƯNG Claude Code là Agentic Workflow tự sửa lỗi
  > Loop 4: [Context: AI gặp lỗi khi thu thập data] → NHƯNG nó không bỏ cuộc, thử 3 cách khác
  > Loop 5: [Context: case study chạy ngon] → NHƯNG có giới hạn cần biết (laptop phải mở, permissions)
  > Loop 6: [Context: mỗi session AI không nhớ] → NHƯNG bạn có thể dạy nó nhớ bằng Memory Log

### 2. 🎵 Rhythm
- **Phân bố câu:**
  - Câu ngắn (2-5 từ): ~25 câu ("Bùm.", "Đơn giản vậy thôi.", "Mất 30 phút mỗi lần.")
  - Câu trung bình (8-15 từ): ~35 câu
  - Câu dài (20+ từ): ~15 câu
- **Jagged edge test:** ✅ Đạt — Mỗi câu 1 dòng riêng, cạnh phải răng cưa rõ

### 3. 🗣️ Tone
- **Ngôn ngữ conversational:** "mình", "bạn", "nghe có vẻ", "đúng không?", "Bùm"
- **Rhetorical questions:** 3 câu ("Nghe có vẻ là hạn chế lớn đúng không?", "Nhưng nó hoạt động thế nào trong thực tế?")
- **Interaction cues:** "Đây là setup của mình", "Hãy xem"
- **Ngôn ngữ lớp 5:** Thuật ngữ kỹ thuật luôn được giải thích ngay (Agentic Workflow → "nó biết tự sửa lỗi", Scheduled Tasks → "lên lịch để AI tự chạy", stateless → "mỗi lần chạy là phiên hoàn toàn mới, không nhớ gì")

- **Tại sao unique:** Đa số video tech sẽ demo tính năng → mình kể từ góc "thuê nhân viên AI" — gần gũi hơn với audience chủ doanh nghiệp. So sánh trực tiếp giữa script cứng vs agent tự chủ tạo "aha moment".
