# 3 điều AI tự động làm thay bạn trong Claude Code (mà không cần bấm nút)

Bạn đang làm 10 việc mỗi ngày.
Trong đó có ít nhất 3 việc AI làm thay bạn được -- nếu bạn biết cách setup.

Claude Code vừa có tính năng Loop. Đây là 3 việc bạn có thể giao cho nó làm tự động ngay hôm nay:

Việc 1: Kiểm tra email quan trọng
Gõ vào Claude Code: "Mỗi 5 phút, kiểm tra inbox xem có email từ [người gửi] không. Nếu có, báo mình ngay."
Kết quả: Bạn không cần refresh email liên tục. Claude Code tự canh cho bạn.

Việc 2: Theo dõi deploy hoặc hệ thống
Gõ: "Mỗi 1 giờ, kiểm tra trạng thái deploy. Nếu có lỗi, báo mình."
Kết quả: Bạn yên tâm làm việc khác, AI lo phần giám sát.

Việc 3: Nhắc việc theo giờ
Gõ: "Lúc 3 giờ chiều, nhắc mình gửi báo cáo cho sếp."
Kết quả: Giống đặt hẹn giờ -- nhưng thông minh hơn vì Claude Code hiểu ngữ cảnh công việc bạn đang làm.

Lưu ý quan trọng:
- Loop chỉ hoạt động khi phiên Claude Code còn mở. Đóng tab = mất loop.
- Mỗi loop tự động hết hạn sau 3 ngày.
- Nếu loop chạy quá thường xuyên, giữ output ngắn gọn để tránh context rot (Claude Code bị "quên" vì quá nhiều thông tin).

Muốn huỷ loop? Chỉ cần nói: "Huỷ việc kiểm tra email đi." Claude Code tự hiểu và tắt.

AI không thay thế bạn. Nhưng AI biết cách làm những việc bạn không nên mất thời gian làm.
