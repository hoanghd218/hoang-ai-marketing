--- KNOWLEDGE NUGGETS ---

1. [Framework] - 4 tang cong cu lap lich cua Claude Code
   Claude Code co 4 cach chay tac vu dinh ky: Agent Loops (CLI, gioi han 3 ngay, phai giu terminal mo), Claude Code Desktop Scheduler (khong het han, song sot khi restart, nhung can Desktop mo), GitHub Actions (chay tren ha tang GitHub, khong can may tinh bat, nhung chi cho tac vu lien quan GitHub), va Remote Control (dieu khien tu xa nhung van session-based). Moi cong cu phu hop voi mot loai tac vu khac nhau.

2. [Paradigm Shift] - Agent Loops la cong cu "micro-task", khong phai AI chay 24/7
   Nhieu nguoi hieu sai rang Agent Loops bien Claude Code thanh mot agent hoat dong lien tuc. Thuc te, no chi chay trong mot session duy nhat, tat khi dong terminal, may ngu, hoac sau 3 ngay. Day la cong cu cho cac tac vu lap di lap lai trong mot phien lam viec cu the, khong phai he thong tu dong hoa dai han.

3. [Canh bao] - Dung tin hype: Agent Loops KHONG phai "Claude 2.0"
   Video chi ra rang da co nhieu video sai lech goi Agent Loops la "Claude 2.0", "AI 24/7", hay "noi chuyen voi Claude qua Telegram". Tat ca deu sai. Hieu sai se dan den that vong khi tinh nang khong lam duoc nhung gi duoc quang cao.

4. [Canh bao] - Desktop Scheduler tao session moi moi lan chay
   Khi dung Claude Code Desktop de lap lich, moi lan chay tac vu se tao mot session moi. Neu ban dat tac vu chay moi phut, sau 1 gio ban se co 60 session. Day la dieu can luu y de tranh qua tai he thong.

5. [Step-by-step] - Cach dung Agent Loops: /loop + Khoang thoi gian + Prompt
   Su dung cuc ky don gian: go "/loop", nhap khoang thoi gian (5 phut, 1 gio, moi sang...), roi viet prompt muon Claude thuc hien. Vi du: "/loop moi 10 phut kiem tra trang thai deploy cua website".

6. [So sanh] - CLI vs Desktop: Khi nao dung cai nao
   Agent Loops (CLI) phu hop cho tac vu ngan han trong mot phien lam viec cu the (vi du: kiem tra deployment moi 10 phut hom nay). Desktop Scheduler phu hop cho tac vu dai han, lap lai hang ngay (vi du: bao cao sang moi ngay qua Slack). Biet dung cong cu dung luc la chia khoa.

7. [Canh bao] - Loop se chan cong viec hien tai cua ban
   Khi loop dang chay va ban dang lam viec voi Claude Code trong cung terminal, loop se khong ngat de chay. No se doi ban xong roi moi thuc hien. Nghia la moi thu chay trong cung mot session, co the gay gian doan workflow.
