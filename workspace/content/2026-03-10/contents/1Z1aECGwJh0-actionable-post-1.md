# 5 việc bạn làm được ngay với Google Workspace CLI + Claude Code

Bạn dùng Gmail, Calendar, Drive mỗi ngày. Nhưng bạn đang tự tay làm tất cả — đọc email, sắp lịch, tìm file, soạn báo cáo. Mỗi việc 5-10 phút, cộng lại cả ngày mất vài tiếng.

Google Workspace CLI + Claude Code giải quyết chuyện này. Đây là 5 việc bạn làm được ngay, không cần biết lập trình:

**1. Phân loại email theo độ quan trọng, tạo Google Sheet có tô màu.**

Nói với Claude Code: "Lấy email chưa đọc tuần này, chia thành 3 nhóm: cần trả lời gấp, cần xem, bỏ qua được. Tạo Sheet có màu đỏ-vàng-xanh." Xong trong 30 giây.

**2. Tổng hợp lịch họp trong tuần, tạo Google Slides báo cáo.**

Thay vì ngồi mở Calendar rồi gõ lại từng cuộc họp, Claude Code tự lấy dữ liệu và tạo slide cho bạn. Gửi sếp luôn được.

**3. Tìm email cũ có "proposal" hoặc "invoice", tự soạn follow-up.**

Bạn quên follow-up khách hàng? Claude Code tìm lại toàn bộ email liên quan, tóm tắt nội dung, rồi soạn sẵn email nhắc nhở. Bạn chỉ cần bấm gửi.

**4. Chuẩn bị họp: lấy email của người tham dự, tìm thư trao đổi trước đó, tạo Google Doc tóm tắt.**

Vào họp mà nắm rõ ngữ cảnh — không cần hỏi lại "hôm trước mình bàn gì nhỉ?"

**5. Dọn dẹp Google Drive: tìm file draft, backup, v1, v2 rồi tạo bảng kiểm kê.**

Drive bạn có bao nhiêu file "final-final-v3"? Claude Code quét hết, liệt kê ra Sheet, bạn quyết định giữ hay xóa.

Cài đặt đơn giản: chạy npm install, paste mega prompt mà video hướng dẫn, Claude Code tự setup phần còn lại.

Điều kiện duy nhất: có Claude Code và tài khoản Google. Không cần biết code. Không cần biết dòng lệnh.
