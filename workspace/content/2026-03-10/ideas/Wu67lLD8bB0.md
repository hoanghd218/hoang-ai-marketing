--- TOM TAT ---

Video gioi thieu GWS CLI (Google Workspace Command Line Interface) -- mot cong cu dong lenh moi cua Google cho phep Claude Code truy cap toan bo Google Workspace (Gmail, Drive, Docs, Sheets, Slides, Calendar) chi bang mot giao dien duy nhat. Thay vi goi API phuc tap hoac cai nhieu MCP server, chi can cai mot CLI la xong. Tac gia demo viec tao Google Docs tu transcript YouTube, tu dong phan loai email theo do uu tien, va tao slide deck co brand guidelines. Diem dac biet la ket hop CLI voi cong cu chup man hinh (ChromeDev Tools) de AI tu kiem tra va sua loi visual. GWS CLI hien con pre-v1.0 (chua phat hanh chinh thuc) nhung da co hon 100 skill/recipe co san va se tu dong cap nhat khi Google them API moi.

--- KNOWLEDGE NUGGETS ---

1. [Paradigm Shift] - CLI thay the API: Lenh bash danh bai goi API cho Google Workspace
   Thay vi dung API endpoints phuc tap hoac nhieu MCP configs, GWS CLI gom tat ca Google Workspace vao mot giao dien dong lenh duy nhat. Claude Code chi can chay lenh terminal la tuong tac duoc voi Gmail, Drive, Docs, Sheets, Calendar -- giam thieu context window va do phuc tap.

2. [Step-by-step] - Cai dat GWS CLI trong 5 buoc
   Quy trinh ro rang: (1) Clone GitHub repo, (2) Tao project tren Google Cloud Console, (3) Thiet lap OAuth consent screen, (4) Tao OAuth client ID va tai file JSON, (5) Chay "gws o login" de xac thuc. Lam theo la chay duoc ngay.

3. [Framework] - AI Email Triage tu dong: Lay email > Cham diem > Tu xu ly
   Dung Claude Code + GWS CLI de lay 30 email chua doc, AI cham diem muc do uu tien dua tren ngu canh kinh doanh, tu dong danh dau da doc nhung email diem duoi 5. Giam thoi gian xu ly email hang ngay dang ke.

4. [Paradigm Shift] - Mot giao dien thong tri tat ca (One interface to rule them all)
   Mot CLI duy nhat thay the nhieu API endpoints va MCP configs rieng le. Giam overhead cho context window cua AI, don gian hoa viec bao tri, va tu dong cap nhat khi Google them tinh nang moi.

5. [Framework] - Vong lap kiem tra bang hinh anh (Visual Validation Loop)
   Quy trinh: AI tao noi dung (Google Slides) > Chup man hinh bang ChromeDev Tools > AI doc anh va danh gia > Sua loi > Chup lai. Day la cach giai quyet van de AI khong "nhin thay" output ma no tao ra.

6. [Canh bao] - GWS CLI con pre-v1.0 -- se co loi va thay doi
   Cong cu chua phat hanh chinh thuc, co the yeu cau xac thuc lai nhieu lan, co the co breaking changes. Nhung da dung tot va chi se tot hon. Nen cai ngay de lam quen.

7. [Bai hoc thuc te] - Tai sao AI tao Google Docs tren API trong nhu rac
   Khi dung API truyen thong, Google Docs chi hien thi raw markdown -- khong dinh dang, khong hinh anh, trong rat te. GWS CLI giai quyet van de nay bang cach tuong tac truc tiep voi Workspace nhu nguoi dung that.

8. [Framework] - Xep chong cong cu (Compound Tool Stacking): CLI + Screenshot + Brand Guidelines
   Ket hop GWS CLI (tao noi dung) + ChromeDev Tools (chup man hinh kiem tra) + Brand guidelines (mau sac, logo) de tao output chat luong production. Khong phai mot tool lam tat ca ma la nhieu tool bo sung cho nhau.
